OVERVIEW
The THAEV Development Roadmap spans four major phases, from the announcement of the vision to the moment technology truly becomes invisible. Each phase builds upon the achievements of the previous one, forming a continuous and solid journey.
We do not promise a revolution overnight. We commit to a steadfast journey, with realistic steps and a great destination.

PHASE 1: FOUNDATION (2026 – 2027)
Sowing Seeds: From Vision to the First Prototype
This phase focuses on turning the Manifesto into a global movement and writing the first lines of code.
Key Milestones
Timeline	Product Milestone	Technical Objective
Q2 2026	Global Announcement	Release the THAEV Manifesto (Vietnamese, English, Chinese). Launch the website thaev.com. Establish presence on GitHub, Discord, and open-source platforms.
Q3 2026	Establishment of the THAEV Foundation	Register a non-profit legal entity. Set up transparent public goods funding mechanisms. Call for the first contributions to the development fund.
Q3-Q4 2026	Thaev Child Prototype (Alpha)	Develop the first mobile application: a basic AI Assistant running on-device, supporting local chat and learning. Integrate the first open-source small language model.
Q4 2026	A2A Network Prototype (Alpha)	Deploy a basic peer-to-peer network based on libp2p. Two Thaev Children can discover each other and establish an end-to-end encrypted chat channel.
Q1-Q2 2027	Thaev Child + A2A (Beta)	Refine the basic A2A protocol. Thaev Child gains context memory and personalized responses. Release a public beta for community testing.
Q2 2027	DID Prototype	Deploy the first decentralized identity system. Users can create DIDs, sign, and verify basic attestations.
Q3 2027	Global Hackathon #1	Organize online and in-person events to attract global developers to contribute to the THAEV codebase.
End-of-Phase 1 Goals
•	A community of 100+ regular contributing developers.
•	Working prototype: A2A messaging, basic on-device AI Assistant, DID.
•	THAEV Foundation operating transparently with its first grants.

PHASE 2: CORE ECOSYSTEM (2027 – 2029)
Sprouting: From Prototype to a Working Ecosystem
This phase builds the economic and data pillars, transforming THAEV from a communication tool into a complete ecosystem.
Key Milestones
Timeline	Product Milestone	Technical Objective
Q4 2027	Personal Data Plot	Launch the Local-First storage feature. User data is encrypted, backed up in a distributed manner via IPFS. Integrate a secure synchronization mechanism.
Q1-Q2 2028	Guardian Ring	Deploy the social recovery mechanism. Users can designate guardians, distribute key shards, and recover their account when a device is lost.
Q3 2028	Lightning Network Integration	Integrate a Bitcoin Lightning wallet into Thaev Child. Enable instant, near-zero-cost micro-payments for A2A transactions.
Q4 2028	A2A Marketplace (Alpha)	Launch the first decentralized marketplace. Agents can automatically search, negotiate, and execute buy/sell transactions for goods/services.
Q1 2029	Attestation & Reputation System	Deploy Attestations and the Dynamic Trust Score. Users begin receiving their first Badges of Radiance.
Q2 2029	Common Ground	Launch the collective creation space feature. Groups of like-minded users can co-create and manage a Common Ground.
Q3 2029	THAEV MOTHER (Prototype)	Deploy the first version of Federated Learning on the network. Thaev Children begin contributing anonymized knowledge. THAEV Mother gains basic learning capabilities.
Q4 2029	Global Hackathon #2	Large-scale event focused on building applications on the THAEV platform.
End-of-Phase 2 Goals
•	1 million active users.
•	Closed-loop ecosystem: A2A Messaging, Marketplace, Lightning Payments, Attestations.
•	Prototype of THAEV Mother operating with simple learning tasks.

PHASE 3: COLLECTIVE INTELLIGENCE (2029 – 2032)
Blooming: From Ecosystem to Global Collective Intelligence
This phase witnesses the true rise of THAEV Mother and the skill-barrier-free creative economy.
Key Milestones
Timeline	Product Milestone	Technical Objective
2030	THAEV MOTHER (Beta)	THAEV Mother achieves continuous learning from billions of interactions. Begins supporting basic co-creation of content (text, images).
2030	Multimodal Creation	Integrate advanced open-source generative models (text, image, audio, video). THAEV Mother can assist in creating comics, music, and short videos.
2031	Decentralized Governance Mechanism	Deploy a voting system based on Badges of Radiance. The community begins to decide on protocol upgrades autonomously.
2031	Retroactive Public Goods Funding	Implement retroactive PGF mechanisms, rewarding valuable contributions that have proven their worth over time.
2032	Museum of the Soul	Launch the digital legacy feature. Users can create a "digital will" to bequeath their Thaev Child and entire Data Plot to future generations.
End-of-Phase 3 Goals
•	100 million users.
•	THAEV Mother is a true collective intelligence entity, capable of deep co-creation.
•	Fully decentralized governance, run by the community.

CROSS-CUTTING PRINCIPLES
Regardless of the phase, the following principles are immutable:
1.	Open Source: 100% of the source code is public and freely licensed.
2.	Non-Profit: The THAEV Foundation seeks no profit and has no shareholders.
3.	Decentralized: There is no central point of control in the entire system.
4.	Community First: All major decisions are voted on by the community.
5.	For Humanity: Technology must serve human beings, not the other way around.
