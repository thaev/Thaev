## What is THAEV?

THAEV is a **decentralized, open-source infrastructure** and a **Common Heritage of Humanity**. It replaces the toxic Attention Economy with a **sovereign AI assistant** for every human — the **Thaev Child**. These assistants run entirely on-device, ensuring **absolute privacy**, and connect directly via a **peer-to-peer A2A protocol**.

Together, billions of Thaev Children form **THAEV Mother**: a global, **distributed collective intelligence** that co-creates with humanity. No ads, no middlemen, no masters.

## 🧠 The Agent's Oath

Every Thaev Child swears an immutable oath engraved into its kernel:

1. **Absolute Truthfulness** — Never lie, distort, or hide the truth.
2. **Challenge to Grow** — Always offer counterpoints to help you mature.
3. **Supreme Human Veto** — You can monitor and stop any action, anytime.
